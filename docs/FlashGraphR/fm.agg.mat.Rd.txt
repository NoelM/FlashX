Aggregation on a FlashMatrixR matrix.

Description:

     This function accepts a basic operator and perform aggregation on
     the rows/columns of a FlashMatrixR matrix with the basic operator.

Usage:

     fm.agg.mat(fm, margin, op)
     
Arguments:

      fm: a FlashMatrixR matrix

  margin: the subscript which the function will be applied over.

      op: a basic operator or an aggregation operator

Value:

     a FlashMatrixR vector.
