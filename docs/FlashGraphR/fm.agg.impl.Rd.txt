Aggregation on a FlashMatrixR vector/matrix.

Description:

     `fm.sum' returns the sum of all the values in the input
     vector/matrix.

Usage:

     fm.sum(m)
     
     fm.min(m)
     
     fm.max(m)
     
Arguments:

       m: The input vector/matrix

Details:

     `fm.min' returns the minimum of all values in the input
     vector/matrix.

     `fm.max' returns the maximum of all values in the input
     vector/matrix.

Value:

     an integer if the input is an integer vector/matrix; a numeric
     value if the input is a numeric vector/matrix.
