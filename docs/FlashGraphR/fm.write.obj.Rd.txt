Write a FlashMatrixR object (vector/matrix) to a file

Description:

     Write a FlashMatrixR object (vector/matrix) to a file

Usage:

     fm.write.obj(fm, file)
     
Arguments:

      fm: a FlashMatrixR object.

    file: a file in the local filesystem.

Value:

     a logical value. True if the object is written to a file
     successfully. Otherwise, FALSE.
