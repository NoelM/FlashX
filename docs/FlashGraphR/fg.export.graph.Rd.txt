Export graph image.

Description:

     This function exports a graph image in FlashGraphR to the local
     filesystem.

Usage:

     fg.export.graph(graph, graph.file, index.file)
     
Arguments:

   graph: The FlashGraphR object

graph.file: The graph file in the local filesystem to which the
          adjacency lists of the graph is exported to.

index.file: The index file in the local filesystem to which the index
          of the graph is exported to.

Value:

     true if the graph is exported to the local filesystem correctly;
     false, otherwise.
