Miscellaneous Mathematical Functions

Description:

     `abs(x)' computes the absolute value of x, `sqrt(x)' computes the
     square root of x.

Usage:

     fm.abs(m)
     
     fm.sqrt(m)
     
Arguments:

       m: a numeric vector or matrix.

Value:

     `abs' returns a vector/matrix of the same type as the input
     vector/matrix; `sqrt' returns a numeric vector/matrix.
