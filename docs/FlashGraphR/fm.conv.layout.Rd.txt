Convert the data layout of a FlashMatrixR matrix.

Description:

     A FMR matrix can store elements in a row-major or column-major
     order. By changing the data layout, we can improve efficiency of
     some matrix operations.

Usage:

     fm.conv.layout(fm, byrow = FALSE)
     
Arguments:

      fm: a FlashMatrixR matrix

   byrow: a logical value to determine the data layout of a
          FlashMatrixR matrix.

Value:

     a FlashMatrixR matrix
