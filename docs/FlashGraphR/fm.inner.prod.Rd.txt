Matrix inner product

Description:

     Inner product of a dense matrix with a dense vector/matrix.

Usage:

     fm.inner.prod(fm, mat, Fun1, Fun2)
     
Arguments:

      fm: A FlashMatrixR matrix

     mat: A FlashMatrixR dense matrix.

Value:

     a FlashMatrixR vector if the second argument is a vector; a
     FlashMatrixR matrix if the second argument is a matrix.

     Da Zheng <dzheng5@jhu.edu>
