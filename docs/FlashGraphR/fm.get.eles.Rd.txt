Get a submatrix from a FlashMatrixR matrix

Description:

     `fm.get.rows' gets specified rows in a FM matrix. `fm.get.cols'
     gets specified columns in a FM matrix.

Usage:

     fm.get.cols(fm, idxs)
     
     fm.get.rows(fm, idxs)
     
Arguments:

      fm: A FlashMatrixR matrix

    idxs: an array of column indices in fm.

Value:

     a submatrix
