fm.get.matrixR Documentation

Load a sparse matrix to FlashR.

Description

Load a sparse matrix to FlashR from difference sources.

Usage

fm.get.matrix(fg)

fm.load.matrix(mat, index, t.mat = NULL, t.index = NULL, in.mem = TRUE)

Arguments

fg

A FlashGraph object.

mat.file

The file that stores the sparse matrix.

index.file

The file that stores the index of the sparse matrix.

Details

‘fm.get.matrix’ gets a FlashMatrix matrix that references a matrix represented by a FlashGraph object.

‘fm.load.matrix’ loads a FlashMatrix matrix from files. The matrix in the file is in the FlashMatrix format.

Value

a FlashMatrix matrix.

Author(s)

Da Zheng <dzheng5@jhu.edu>

Examples

fg <- fg.load.graph("graph.adj", "graph.index")
fm <- fm.get.matrix(fg)
