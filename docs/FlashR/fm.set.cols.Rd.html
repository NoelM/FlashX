fm.set.colsR Documentation

Set the specified column of a FlashMatrix matrix.

Description

Set the specified column of a FlashMatrix matrix.

Usage

fm.set.cols(fm, idxs, m2)

Arguments

fm

A flashMatrixR matrix

idxs

an array of column indices in fm.

m2

A flashMatrixR matrix or vector.

Value

a logical value that indicates whether the operation is successful.
