fm.matrixR Documentation

Create a matrix from the given set of values.

Description

The given set has to be a FlashMatrix vector.

Usage

fm.matrix(vec, nrow, ncol, byrow = FALSE)

Arguments

vec

a given vector.

nrow

the number of rows in the output matrix.

ncol

the number of columns in the output matrix.

byrow

logical. If FALSE (the default) the matrix is filly columns, otherwise the matrix is filled by rows.

Value

a FlashMatrix matrix
