fm.agg.matR Documentation

Aggregation on a FlashMatrix matrix.

Description

This function accepts a basic operator and perform aggregation on the rows/columns of a FlashMatrix matrix with the basic operator.

Usage

fm.agg.mat(fm, margin, op)

Arguments

fm

a FlashMatrix matrix

margin

the subscript which the function will be applied over.

op

a basic operator or an aggregation operator

Value

a FlashMatrix vector.
