FlashX is a collection of big data analytics tools. FlashX performs data analytics in the form of graphs and matrices and utilize solid-state drives (SSDs) to scale to large datasets in a single machine. It has three main components: FlashGraph is a general-purpose graph analysis framework that allows users to write graph algorithms to analyze billion-node graphs in a single machine; FlashMatrix is a matrix computation engine with a small set of generalized matrix operations to express varieties of data mining and machine learning algorithms; FlashR is an extended R programming framework to process datasets at a scale of terabytes in parallel.

Performance

Each of FlashX components has lightning-fast speed when they run in memory. While running on SSDs, they have performance comparable to the state-of-art in-memory performance. FlashR has performance comparable to the optimized parallel C code.

Scalability

All FlashX components can scales beyond memory capacity. They process graphs with hundreds of billions of edges or even trillions of edges or datasets with terabytes in a single commodity machine.

Flexibility and ease of programming

FlashR provides a unified R programming interface for users. It mimics the original R programming environment and also provides a set of generalized matrix operations to express varieties machine learning and data mining algorithms. Users do not need to explicitly handle parallelization and external-memory access.

FlashR

An extension of the R programming framework to process datasets at a scale of terabytes with the speed of optimized C code. FlashR integrates FlashMatrix with R. It provides users a familiar R programming environment as well as generalized matrix operators to express many machine learning algorithms completely in R.
